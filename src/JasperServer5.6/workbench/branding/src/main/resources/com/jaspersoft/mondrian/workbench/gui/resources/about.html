Jaspersoft OLAP Schema Workbench


Version : 4.0.0

The Jaspersoft OLAP Schema Workbench allows the visual creation, validation and testing of configuration of OLAP cubes for the Mondrian OLAP engine. OLAP schemas managed in the Workbench can be uploaded to a Jaspersoft OLAP server.


Documentation for the details of schemas is included in the "docs" directory of the Workbench installation.


Using the Workbench :
  1. Have a JDBC driver JAR for your database installed in the workbench under the "drivers" directory of the Workbench installation.
  2. Define and test a database connection to your data warehouse.
  3. Review the structure of your data warehouse with the JDBC Explorer.
  4. Create a schema in the Schema Explorer, selecting your database connection.
    • Add schema elements with context sensitive popup menus
    • Drop downs show relevant values from the database and schema
    • Syntax problems are highlighted
  5. Test the schema against the database from the schema editor.
  6. Open the MDX Query window and generate default MDX queries.
  7. Run and save MDX queries, which can be used in Jaspersoft OLAP Analysis Views.
  8. In the Jaspersoft OLAP service, configure the database connection, OLAP schema, Mondrian connection and Analysis Views.


New features in this version:
  • Upgrade to Mondrian 3.2.0


System Requirements : Java 1.5 and above
